Proyect Description

The expense controller project is a project implemented using React, coupled with Vite and written with TypeScript. For the styling of the components we used Tailwind and certain special components like the modal window we used an additional library called HeadlessUi, while for the icons we used the HeroIcons library. For the dynamic graphic that we have in the project we worked with a react library called react-circular-bar and like all the logic of this application was handled with a useReducer for the general states, while for specific states we used useState, for certain validations we used the Hook of useMemo, while for the localStorage function we used the hook of useEffect. The handling of our dates was handled with React-date-picker, all this was built applying context API to be able to handle our useReducer in a global way. The purpose of the project is to be able to have a visual and written control of our expenses and categorize it to be able to see in what we are spending the money and in what categories we could make cuts if we are not reaching the end of the month with the current expenses.
